== Installation ==
This section describes how to install the plugin and get it working Best Way.
e.g.
1. Upload `wpsection` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Start using it by going to the elementor editor.
